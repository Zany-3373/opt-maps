  <h1>経路最適化ツール</h1>
  <p>リンク：https://zany-3373.github.io/opt-maps/</p>
  <p>・GoogleマップAPIを利用して複数の経由地を最も効率的に巡るルートを提示します。<br>
  ・提示された経路を利用してサイトからGoogleマップに遷移し、ルート案内を開始できます。<br>
  ・サイト上では出発地と目的地を含む合計25地点（経由地上限23地点）まで追加できます。<br>
  ・Googleマップ上でのルート案内は最大10地点までであるため、それを超過する最適化経路をGoogleマップで開くと、経由地が最適化された順に8地点（出発地と目的地が同一の場合は9地点）まで案内されます。
  </p>
